Code Tokenizer

El propósito de esta página es facilitarte el trabajo con ChatGPT:,
para aquellos casos en los que tu código exceda el número máximo de tokens del chat ( 2048 )
  • 1. Selecciona el archivo basado en texto que quieres procesar - .txt, .js, .md, .py, etc...
  • 2. Presiona el botón "Procesar"
  • 3. Espera a que se descarguen los archivos procesados - cada uno estará dentro del límite máximo de tokens del chat
  • 4. ¡Listo! Ya puedes subir el archivo procesado a ChatGPT
