Code Tokenizer

El propósito de esta página es facilitarte el trabajo con ChatGPT:,
para aquellos casos en los que tu texo o código exceda el número máximo de tokens del chat ( 2048 )
  • 1. Selecciona el archivo (o carpeta con archivos) que quieres procesar - .txt, .js, .md, .py, etc...
  • 2. Los archivos serán serán divididos en trocitos, cada uno dentro del límite máximo de tokens del ChatGPT
    • 3.a - Puedes copiar cada trozo al portapapeles presionando el botón "Copiar al portapapeles". El botón mostrará el porcentaje de partes copiadas
    • 3.b - Presiona el botón "Procesar" para descargar los trocitos y usarlos más adelante
  • 4. ¡Listo! Ya puedes subir por orden cada parte copiada a ChatGPT
